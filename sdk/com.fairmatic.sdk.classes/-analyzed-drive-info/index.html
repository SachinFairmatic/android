AnalyzedDriveInfo

class AnalyzedDriveInfo : DriveInfo

Completely analyzed information about a drive recorded by the Fairmatic SDK. The drive may have started manually by the application or due to Fairmatic auto drive detection. This is available with FairmaticBroadcastReceiver.onDriveAnalyzed callback.

This is available only when FairmaticDriveType is not equal to FairmaticDriveType.INVALID. The info provided in this object supercedes the info provided in EstimatedDriveInfo via the FairmaticBroadcastReceiver.onDriveEnd callback for the same DriveInfo.getDriveId. The application should ideally update the info stored for this drive with the info provided here.
