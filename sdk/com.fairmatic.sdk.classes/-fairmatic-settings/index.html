FairmaticSettings

class FairmaticSettings(errors: List<FairmaticSettingError>, warnings: List<FairmaticSettingWarning>)

This class surfaces errors in device or application settings that affect SDK trip detection as well as warnings that may affect the normal functioning of the Fairmatic SDK.

Parameters

errors

List of errors that must be resolved for trip detection to work correctly.

warnings

List of warnings that may affect the normal functioning of the Fairmatic SDK.
