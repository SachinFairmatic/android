setEnabledBluetoothDriveStart

fun setEnabledBluetoothDriveStart(enabledBluetoothDriveStart: Boolean)

Set this property to true if the app supports Bluetooth Drive Start. If set to true, the application will receive FairmaticBroadcastReceiver.onDriveStart callback whenever a successful bluetooth connection is established with an already associated vehicle. If the vehicle is not already associated, call FairmaticVehicleTagging.associateVehicle before connecting the device to the vehicle via Bluetooth. This feature needs FairmaticDriveDetectionMode to be FairmaticDriveDetectionMode.AUTO_ON.

Default value is false. Please contact fairmatic Support in case the feature is not enabled.
