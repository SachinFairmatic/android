startDriveWithPeriod1

fun startDriveWithPeriod1(context: Context, callback: FairmaticOperationCallback?)

Start a drive in Fairmatic insurance period 1. A manual trip with trackingId com.fairmatic.sdk.internal will be started immediately.

The entire duration in this period will be recorded as a single trip. If period 1 is already in progress with the same trackingId, this call will be a no-op.

Parameters

context

Context - Application context

callback

FairmaticOperationCallback - called with the result of the operation
